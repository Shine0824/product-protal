📋 猪实验卡片

  • 与其他物种（家禽/反刍...）共用同一卡片列表
  • 卡片左侧色条表示状态（绿=正常，橙=预警）
  • 每张卡片展示：编号 + 物种 + 阶段 + 完整度 + 异常等级
  • 卡片主体点击 → 实验数据报告（P9）
  • 右上角"去录入" → 根据当前步骤跳转对应阶段录入页
  • 顶部统计胶囊快速筛选

💡 关键交互

卡片右上角"去录入"是核心入口。
点击后自动带入实验编号 + 当前步骤，
跳转到对应的流程页面（分组/巡栏/结束/归档）。

🎯 栏位锚定 + 模块漂移

  • 栏位选择器固定在页面顶部，作为"锚点"
  • 切换栏位 → 模块Tab不变，表单刷新
  • 切换模块Tab → 栏位不变，表单切换
  • 绿点 = 当前栏位当日该模块有记录

⚡ 操作节奏

栏1 → 药品Tab(录入) → 疫苗Tab(录入) →
切栏2 → 药品Tab(录入) → 环控Tab(录入) →
切栏3 → ...
