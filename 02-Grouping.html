🎯 页面目标

实验分组页用于在实验开始前将试验动物按处理组、区组、栋舍、单元、栏位、品系、耳标等维度进行系统化分配，确保实验数据的可追溯性和统计分析的合理性。

👤 角色与权限

  • 项目负责人：可创建、修改、提交分组
  • 实验员：可录入基础数据，提交后不可修改
  • 审核员：可审阅分组合理性并签字确认

🧭 操作流程

  1. 选择处理组（饲料配方/营养策略）
  2. 配置区组（随机区组/拉丁方等）
  3. 绑定栋舍-单元-栏位的物理位置
  4. 录入品系、耳标、性别等基础属性
  5. 录入初始体重、头数、断奶时间
  6. 下载模板/上传 Excel 批量导入
  7. 校验通过后提交，进入"日常记录"阶段

⚠️ 注意事项

分组一旦提交，不可回退修改。如需调整，需走"作废 → 重新分组"流程，并保留审计日志。

初始头数(只)整数栏位头数

📐 计算字段

  • 初始只均重 = 初始称重 / 初始头数
  • 初始头重 = 初始只均重 × 头数
  • 初始余料 = 上阶段余料（来自前序阶段）

⚙️ 关键业务规则

  • 处理组均衡：每个处理组至少 3 个重复（栏位）
  • 区组随机：同一区组内各处理组出现一次
  • 体重 CV ≤ 5%：同一处理组内初始体重变异系数不超过 5%
  • 性别配比：公母比例需在 1:1 ± 10% 范围内
  • 断奶日龄：保育阶段 21-28 日龄，育肥阶段 ≥ 70 日龄

🔁 状态机

草稿(Draft) → 校验中(Validating) → 已提交(Submitted) → 已审核(Approved)
   ↓                ↓                      ↓
作废(Voided)   校验失败(Invalid)     已驳回(Rejected)

🧮 统计指标

  • 各处理组均重、SD、CV
  • 区组效应方差分析（ANOVA）
  • 处理组间 t 检验 / Duncan 多重比较

📐 前端校验

字段规则
耳标序号唯一性校验（栏内+跨栏）
初始称重> 0 且 ≤ 头数 × 50kg
初始头数> 0 整数
断奶时间≤ 开始时间
栋-单元-栏位组合唯一性
处理+区组组合唯一性

🔄 后端校验（建议）

  • 与上一实验的栏位占用冲突检查
  • 栋舍容量上限校验（基于头数 × 占地面积）
  • 品系与生长阶段适配性检查
  • 耳标号与公司资产库比对

📦 数据结构

Grouping {
  id: string
  experimentId: string
  treatment: number      // 处理组
  block: number          // 区组
  building: string
  unit: string
  pen: string
  strain: string
  earTagColor: string
  earTagNo: string
  gender: 'M' | 'F' | 'C'
  weaningDate: string
  weaningAge: number
  startDate: string
  initialAge: number
  initialWeight: number  // 整栏重
  initialHeadWeight: number
  initialHeadCount: number
  initialAvgWeight: number
  initialRemainingFeed: number
  status: 'draft' | 'submitted' | 'approved'
}

📡 事件

  • grouping:created - 创建分组
  • grouping:submitted - 提交分组（触发阶段流转）
  • grouping:approved - 审核通过
