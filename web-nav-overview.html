🏗️ Web外壳架构 (index-web.html)

  • 左侧栏 (220px) — 深蓝渐变背景，三级导航菜单
  • 顶部栏 (50px) — 白色背景，显示当前页面标题
  • iframe 内容区 — 加载 00~10 号独立HTML页面
  • 纯前端实现，点击导航 → 切换 iframe src
  • 每个子页面 (00~10) 自带独立的侧边栏 + 顶部导航（嵌入在 iframe 内时与外壳侧边栏形成双重导航）

⚠️ 注意

00~10 各页面内部也包含独立的侧边栏与顶部导航（作为独立页面打开时有完整布局）。
当嵌入 index-web.html 的 iframe 时，子页面自身的 sidebar 仍在 iframe 内可见，形成双层导航结构。
建议：通过 index-web.html 外壳访问时，子页面可考虑隐藏自有导航，仅保留内容区。

F 技术要点与注意事项

🏗️ 架构特点

  • iframe 沙箱隔离 — 各子页面独立运行，互不影响
  • 双层导航 — 外壳有侧边栏，子页面 (00~10) 内部也有独立侧边栏
  • Element Plus — 子页面使用 CDN 引入 Element Plus 组件库
  • Vue 3 (App端) — index-app.html 使用 Vue 3 CDN 构建 SPA
